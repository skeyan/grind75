Two Sum

Scan once while storing each value’s index in a hash map; for each nums[i], if target - nums[i] is already in the map, return the two indices. This turns the complement lookup into O(1) per step.

Valid Parentheses

Use a stack of opening brackets; on a closing bracket, pop and check it pairs with the top. If the stack is wrong or nonempty at the end, the string is invalid.
