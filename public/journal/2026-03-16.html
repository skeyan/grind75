Valid Anagram

Count character frequencies for both strings (or sort both) and check the counts or sequences match. A fixed-size array works for lowercase letters.

Binary Search

Keep left and right bounds; compare mid to the target and discard the half that cannot contain the answer until you find the index or the range empties.

Flood Fill

Start from the given cell; DFS or BFS into neighbors that still have the original color, recoloring as you go and marking visits so you don’t loop.
