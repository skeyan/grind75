Lowest Common Ancestor of a Binary Search Tree

Walk from the root: if both p and q are smaller than the current node go left; if both are larger go right; otherwise the current node splits them and is the LCA.

Maximum Subarray

Kadane’s idea: for each position, either extend the previous subarray sum or start fresh at the current element; track the maximum sum seen along the way.
