Binary Tree Inorder Traversal

Iterative DFS walks left as far as possible while pushing ancestors, then pops to emit the node value and steps into the right subtree. When the current pointer is null but the stack is not, each pop resumes the next step of left-root-right order.

Binary Tree Postorder Traversal

One stack walks the left spine; then the algorithm peeks at the top node and either descends into an unfinished right subtree or pops and records the value, using lastVisited so it does not treat a completed right child as new work. That yields left-right-root without recursion.
